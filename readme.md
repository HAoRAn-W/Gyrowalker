# Gyrowalker
A embbed project to measure distance with gyroscope
## Functions
walk in straight line

support different speed

calculate distance

LCD display(optional) to show distance

## Hardware & Framework
Board: STM32F429I

Gyroscope: I3G4250D

Framework: Mbed
